# Jaswanth Kumar | AI Engineer Portfolio

A premium, high-performance portfolio website built with React, TypeScript, and Framer Motion. This project features immersive AI-themed animations, ultra-smooth transitions, and a modern, responsive design tailored for a professional AI & Cloud Engineer.

![Portfolio Preview](public/og-image.png)

-   **3D Interactive Elements**: Project cards with perspective-shifting tilt effects that respond to mouse movement.
-   **Neural Scroll Pulses**: One-shot, scroll-triggered activation effects (neural pulses and corner accents) as you explore the site.
-   **Responsive & Optimized**: GPU-accelerated transitions and optimized Canvas rendering ensure 60FPS performance across desktop and mobile.
-   **Dynamic Navigation**: Custom-built navbar with active section tracking, smooth anchor scrolling, and interactive social icons.

Follow these steps to set up the project locally:

